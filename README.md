# Cover Make

一个基于 Web 的封面图片生成工具，支持多种平台尺寸、丰富的背景效果、自定义文字和图标，帮助用户快速创建精美的博客封面、社交媒体封面等。

## ✨ 功能特性

### 🎨 背景效果
- **渐变背景**：支持线性、径向、圆锥渐变，可自定义多个颜色节点和角度
- **纯色背景**：简洁的纯色填充
- **自定义图片**：支持上传本地图片作为背景，可调节模糊度和暗化效果
- **网格渐变**：基于多色网格的流体渐变效果
- **噪点纹理**：程序化生成的噪点纹理背景
- **随机配色**：一键生成和谐的随机配色方案

### 📝 文字编辑
- **三层文字**：主标题、副标题、作者名，独立控制
- **丰富字体**：集成 16+ 款中文字体（思源黑体、站酷系列、霞鹜文楷、得意黑等）
- **自由拖拽**：画布上直接拖拽调整文字位置
- **自定义文字**：支持添加多个自定义文字元素

### 🎯 图标管理
- **SVG 图标**：支持粘贴 SVG 代码或上传 .svg 文件
- **图标容器**：支持纯色/毛玻璃背景样式，圆角矩形、圆形、菱形三种形状
- **阴影与边框**：可配置投影效果和自定义边框
- **自由拖拽**：画布上直接拖拽调整图标位置

### 📐 平台预设
内置 6 种常用平台尺寸：
| 平台 | 尺寸 | 比例 |
|------|------|------|
| 博客封面 | 1200×675 | 16:9 |
| 公众号 | 900×500 | - |
| 小红书 | 800×800 | 1:1 |
| 视频封面 | 1280×720 | 16:9 |
| PPT 封面 | 1920×1080 | 16:9 |
| 抖音/竖版 | 1080×1920 | 9:16 |

### 🎭 预设模板
内置 6 种风格模板：科技风、文艺风、商务风、极简风、暗黑风、清新风，一键应用。

### 💾 配置管理
- **导出配置**：将当前所有设置保存为 JSON 文件
- **导入配置**：恢复之前保存的配置
- **快捷键支持**：
  - `Ctrl/Cmd + S`：导出配置
  - `Ctrl/Cmd + D`：下载封面
  - `Ctrl/Cmd + R`：重置所有设置

### 🌐 多语言
支持中文和英文界面切换。

## 🚀 使用方法

### 本地使用
直接在浏览器中打开 `index.html` 即可使用，无需任何构建步骤。

### 在线访问
通过 GitHub Pages 部署后访问：
```
https://你的用户名.github.io/cover-make/
```

## 📥 下载封面

支持多种导出格式：
- **PNG**：无损压缩，透明背景（推荐）
- **JPEG**：有损压缩，体积较小
- **WebP**：现代格式，压缩率高
- **AVIF**：最新格式，体积最小

## 🛠️ 技术栈

- **纯前端实现**：HTML5 Canvas + Vanilla JavaScript + CSS3
- **无依赖**：零第三方库，轻量高效
- **响应式设计**：适配桌面端和移动端

## 🎨 添加图标

1. 访问 [yesicon.app](https://yesicon.app) 搜索需要的图标
2. 点击图标 → 复制 SVG 代码
3. 在 Cover Make 的"图标"面板中粘贴 SVG 代码，点击添加
4. 或直接上传本地的 `.svg` 文件
